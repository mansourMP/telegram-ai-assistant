# Code of Conduct

This project adopts a simple standard for public collaboration:

- Be respectful.
- Be precise.
- Critique code and ideas, not people.
- Prefer reproducible reports over vague complaints.
- Do not post secrets, credentials, or private personal data in issues or pull requests.

Unacceptable behavior includes harassment, insults, discrimination, or intentionally disruptive contributions.

If a contribution is not constructive or safe for the project, it may be closed or rejected.
