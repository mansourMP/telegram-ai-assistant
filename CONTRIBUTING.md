# Contributing to Mansur Bot

Thank you for your interest in contributing!

## Setup

1.  Fork the repo and clone it.
2.  Install dev dependencies:
    ```bash
    make install
    ```
3.  Set up your `.env` file (see `.env.example`).

## Guidelines

- **Code Style**: We follow PEP 8. Please run `black .` before committing.
- **Tests**: Add tests for new features. Run `make test` to verify.
- **Commits**: Use clear, conventional commit messages.
- **Pull Requests**: Please describe your changes clearly.

## Reporting Issues

If you find a bug, please open an issue with reproduction steps.
